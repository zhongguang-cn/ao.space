## Source code download

请按顺序执行一下命令，下载整个项目源码:

- 创建本地目录， 执行命令: `mkdir ao.space`
- 进入项目目录，执行命令: `cd ao.space`
- 初始化，执行命令: `git submodule init`
- 下载源码，执行命令: `git submodule update`
